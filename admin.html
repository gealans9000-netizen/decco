📝 코드 업데이트 가이드

새 코드 받았을 때 업데이트 순서:

  1. 위 "📤 파일 업로드" 버튼 클릭 → GitHub Upload 페이지 열림
  2. 새 파일을 끌어다 놓기 (또는 "choose your files")
  3. 화면 아래로 스크롤 → "Commit changes" 클릭
  4. 1~2분 대기 (Vercel 자동 재배포)
  5. 사이트에서 Ctrl + Shift + R (강제 새로고침)으로 확인

💡 작은 수정이라면 "✏ 편집" 버튼으로 GitHub에서 직접 코드 수정도 가능합니다.

ℹ 안내

• 역할 변경: 드롭다운으로 즉시 변경됩니다 (재로그인 필요).

• 새 사용자 추가 / 사용자 삭제: 위 "👥 사용자 관리" 클릭하면 Supabase에서 처리.

• 본인 비밀번호 변경은 메인 화면 우측 상단 ⚙ → "비밀번호 변경"을 사용하세요.
